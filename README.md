## Installation

### RubyGems

```
$ gem install fluent-plugin-redis-streams
```

### Bundler

Add following line to your Gemfile:

```ruby
gem "fluent-plugin-redis-streams"
```

And then execute:

```
$ bundle
```

## Configuration

You can generate configuration template:

```
$ fluent-plugin-config-format output redis-streams
```
